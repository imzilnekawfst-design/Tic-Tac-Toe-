# Tic-Tac-Toe Game

## Description
Un jeu classique **Tic-Tac-Toe** développé en Java.  
- Interface graphique avec **Java Swing**  
- Deux joueurs peuvent jouer sur le même ordinateur  
- Affiche le ( winner et Loser )

## Technologies
- Java (VS CODE )  
- Java Swing pour l’interface graphique
